**Arcane Memory** is a high-fidelity, fantasy-themed memory match game built with React and Tailwind CSS. Step into a world of magic where your memory is your strongest weapon. Navigate through levels of increasing difficulty, manage your Arcane Energy, and unlock powerful skills to clear the grid before the time runs out.

## ğŸš€ Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (Version 14 or higher recommended)
-   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/naitik09090/Memory-Match-Game.git
    cd Memory-Match-Game
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in browser**:
    Navigate to `http://localhost:3000` (or the port shown in your terminal).

## ğŸ® How to Play

1.  **Start Quest**: Click the "START QUEST" button from the main menu.
2.  **Match Pairs**: Click two cards to flip them. If the icons match, they stay up!
3.  **Manage Time**: Watch the timer at the top. If it hits zero, your quest ends.
4.  **Use Skills**: When you have enough Arcane Energy, click the skill icons in the HUD to get an advantage.
5.  **Clear Stages**: Match all cards on the board to advance to the next level.
